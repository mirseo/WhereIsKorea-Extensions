개인정보 처리방침

KoreaSelector는 사용자의 개인정보를 보호합니다.
이 페이지에서 개인정보 처리에 대해 자세히 알아보세요!

이 확장프로그램은 개인정보를 수집하지 않아요!

KoreaSelector는 사용자의 개인정보를 전혀 수집, 저장, 전송하지 않는 100% 로컬 확장 프로그램입니다.

1. 개인정보의 수집

KoreaSelector는 어떠한 개인정보도 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보 수집 없음
  • 브라우저 사용 패턴이나 방문 기록 추적 없음
  • 사용자의 입력 데이터 저장 없음
  • 쿠키나 로컬 스토리지 사용 없음

2. 데이터 처리 방식

확장 프로그램의 모든 작업은 사용자의 브라우저 내에서만 처리됩니다:

  • 웹 페이지의 국가 선택 필드 탐지
  • 대한민국 관련 옵션 식별 및 자동 선택
  • 모든 처리는 로컬에서 실행되며 외부 서버로 데이터 전송 없음

3. 권한 사용

KoreaSelector가 요청하는 권한과 사용 목적:

activeTab

현재 활성화된 탭에서 국가 선택 필드를 찾기 위해 사용

storage

사용자의 on/off 설정을 로컬에 저장하기 위해 사용

tabs

탭 정보 접근을 위해 사용 (개인정보 접근 없음)

4. 제3자와의 정보 공유

KoreaSelector는 어떠한 데이터도 제3자와 공유하지 않습니다.

  • 광고 네트워크나 분석 도구 사용 없음
  • 외부 서버로의 데이터 전송 없음
  • 사용자 추적이나 프로파일링 없음

5. 데이터 보안

데이터를 수집하지 않기 때문에 보안 위험이 최소화됩니다:

  • 로컬에서만 동작하는 완전한 오프라인 처리
  • 오픈소스로 코드 투명성 보장
  • 정기적인 보안 업데이트 제공

6. 문의 및 연락처

개인정보 처리방침에 대한 문의사항이 있으시면:

  • GitHub 저장소의 Issues 섹션을 통해 문의
  • 개발자: mirseo
  • 저장소: WhereIsKorea-Extensions

7. 개인정보 처리방침 변경

이 개인정보 처리방침은 필요에 따라 업데이트될 수 있습니다:

  • 중요한 변경사항은 GitHub 저장소를 통해 공지
  • 사용자의 권리에 영향을 주는 변경은 사전 고지
  • 최신 버전은 항상 이 페이지에서 확인 가능
